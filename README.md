# 🎬 영화 관람권 구매 사이트 및 관리자 페이지

📌 **영화 관람권 구매 및 관리 기능을 제공하는 웹사이트**  
사용자는 원하는 영화를 선택하고 관람권을 구매할 수 있으며, 관리자는 영화 정보를 저장하고 관리할 수 있습니다.

### 🎛 관리자 페이지 (영화 정보 저장)

- 영화 정보 수정 및 삭제 ✏️
- CSV 파일 다운로드 📂
